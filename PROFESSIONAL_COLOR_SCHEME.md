## Color Palette:

### Primary Colors:
```css
Dark Red:    #8B0000  /* Main buttons, titles, accents */

Dark Gray:   #2F2F2F  /* Headers, professional elements */
```

### Usage:
- **Dark Red (#8B0000):** Primary buttons, main titles, important elements
- **Olive Green (#556B2F):** Secondary buttons, gradient backgrounds
- **Dark Gray (#2F2F2F):** Table headers, navbar elements
- **Black (#000000):** Body text, borders
- **White (#FFFFFF):** Card backgrounds, button text
